# Presentation Figures

This folder contains ready-to-use diagram images for report/presentation.

- SVG:
- `docs/figures/block_diagram_presentation.svg`
- `docs/figures/pin_allocation_overview.svg`
- `docs/figures/waterfall_plan_to_final.svg`

- PNG:
- `docs/figures/block_diagram_presentation.png`
- `docs/figures/pin_allocation_overview.png`
- `docs/figures/waterfall_plan_to_final.png`

Regenerate PNG:

`python docs/figures/generate_presentation_pngs.py`

Recommended usage:
1. Use `block_diagram_presentation.svg` in the architecture/system overview slide.
2. Use `pin_allocation_overview.svg` in hardware design/circuit slide.
3. Use `waterfall_plan_to_final.svg` in the project timeline/process slide.
